# Dice Throne Companion App

React + TypeScript + Vite rebuild of the Hi-Fi design (`../Dice Throne Companion - Hi-Fi.dc.html`) — see the top-level `../README.md` for the full screen/behavior spec and `../DEVLOG.md` for progress.

## Development

```
npm run dev     # regenerates src/data/constants.ts from the design file, then starts Vite
npm run build   # type-checks, regenerates constants.ts, then builds
npm run lint    # oxlint
```

`src/data/constants.ts` is generated by `extract.cjs` from the `.dc.html` design source — don't hand-edit it, edits get overwritten on the next `dev`/`build`.
